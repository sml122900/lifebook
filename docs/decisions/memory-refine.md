# 결정 — 문장 다듬기 (원문 비파괴 AI 교정)

2026-06-12. MVP(맞춤법만) → Lv2(문장 정리 통합) 같은 날 완성.

## Problem

어르신 회상은 음성 입력·입말이 많아 군말(어/음/그니까)·오타·띄어쓰기가
섞인다. 가족에게 보여줄 글로 다듬고 싶지만:

- **원문이 자산** — 말투·사투리·입버릇이 그 사람 자체. AI 가 덮어쓰면
  복구 불가 손실.
- AI 창작 금지 원칙(북극성) — 없는 내용·감정을 붙이면 회상이 아니라
  소설이 된다.
- 시니어가 "AI 가 내 글을 바꿨다" 를 인지·통제할 수 있어야 한다.

## Action

### 결정 1 — 원문 영구 보존: 별도 컬럼 + 표시 게이트

`UserMemory` 에 `refinedText`/`refinedAt`/`displayRefined` 3컬럼 추가
(content 덮어쓰기 없음). 교정 직후엔 저장만 — **사용자가 전/후 카드를
보고 [이대로 바꾸기] 를 눌러야** `displayRefined=true` 로 표시 전환.
전환 후에도 "원래 글 보기" 토글 상시. content 수정 시 3필드 초기화
(life-events 는 실제 content 변경 시에만 — 장소·제목만 고치면 교정본
보존. era-stash 는 content 항상 교체라 무조건).

### 결정 2 — 표시 스왑은 읽기 헬퍼 한 곳에서

`getLifeEvents` 가 `displayRefined && refinedText ? refinedText :
content` — 연혁·상세·비서 컨텍스트 등 호출자 0줄 자동 적용.
`getLifeEventById`(편집 화면)는 **원문 유지** + refined 필드 별도 노출
— 편집 textarea 가 교정본을 또 편집하는 사고 차단. 가족 룸
(`listRoomMemories`)은 별도 select 라 원문 유지 — 룸 적용은 후속 결정.

### 결정 3 — 무료 (Haiku, 차감 0)

`chargeOneShot` 호출 없음. 교정은 회상 작성을 돕는 기본기 — 토큰
장벽을 두면 가장 필요한 사용자(입말 많은 어르신)가 안 쓴다. 비용은
Haiku 단가로 운영 흡수 (voice-cleanup 과 같은 톤).

### 결정 4 — 길이 가드 0.6~1.2 (서버측 왜곡 방어)

프롬프트 금지 조항만으로는 요약·살 붙이기를 못 막는다. 결과 길이가
원문의 60~120% 를 벗어나면 저장 없이 no_change. 하한이 0.8 이 아니라
**0.6 인 이유**: 군말 많은 입말은 제거만으로 60%대까지 떨어지는 게
정상(실측 67%) — 0.8 은 정확히 이 기능이 필요한 글을 과차단했다
(상세: `docs/troubleshooting/refine-length-guard-filler.md`).

### 결정 5 — 사투리는 명시 예시로 보호

"사투리 제거 금지" 추상 조항만으로는 모델이 "어머이"→"어머니" 표준화
(실측). 예시 열거(어머이·어무이·억수로·~했당께·~카더라·그라모) +
"군말(어, 음, 그니까, 인제, 막)과 사투리는 다르다 — 군말만 제거" 대비
조항으로 보강 후 보존 확인.

## Result

- 마이그 1건(ADD COLUMN 3줄, 기존 행 무영향), 새 모델 0.
- 실측: 군말 표본 123자 → 94자(76%), 사투리 3종 보존 + 군말 전부 제거.
- tsc 0, test-life-events(7)·test-era-stash(17) 회귀 0.

### 트레이드오프

- 0.6 하한은 장문에서 모델이 일부 문장을 빠뜨려도 통과할 여지 —
  프롬프트 "빼지 마라" 조항 + 실사용 모니터링으로 보완.
- 무료 정책은 남용 표면 — 사용자당 호출 빈도 텔레메트리 후 필요 시
  rate limit.

### 일반화된 학습

1. **AI 가 사용자 콘텐츠를 고칠 땐 비파괴 + 명시 승인 게이트** — 별도
   컬럼·전/후 비교·표시 토글이 한 세트.
2. **프롬프트 금지는 서버 검증과 쌍** — 길이 비율 같은 기계적 가드가
   "지켰는지" 를 코드로 판정한다.
3. **가드 임계값은 실측으로** — 0.8 은 합리적으로 들리지만 도메인
   (군말 많은 입말)을 실측하니 과차단. 표본 1건이 스펙을 고쳤다.
4. **추상 금지 조항은 예시로 고정** — "사투리 보존" 보다 "어머이·억수로
   그대로" 가 모델을 움직인다.
