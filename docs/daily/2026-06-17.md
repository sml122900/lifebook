## 5. 인앱 브라우저 구글 로그인 차단 대응 (`533c32b`)

### 증상

### 해법

`app/login/InAppBrowserGuard.tsx` 신규 (3개 export):

| export | 역할 |
|--------|------|
| `InAppBrowserGuard` | mount 시 UA 감지, Android면 intent URL 발사 |
| `InAppIosBanner` | iOS 인앱 감지 시 amber 배너 (Safari로 열기 유도 + URL 복사) |
| `InAppGoogleNote` | 인앱 공통: 구글 버튼 아래 한 줄 안내 |

**UA 감지**: `KAKAOTALK|Instagram|NAVER|FBAN|FBAV|FB_IAB`

**Android 분기**:
- KakaoTalk: `kakaotalk://web/openExternal?url=…`
- 기타: `intent://{host}{path}#Intent;scheme=https;package=com.android.chrome;end`

**iOS 분기**:
- amber 배너 + "···→Safari로 열기" 안내
- URL 복사 버튼 (clipboard API + execCommand 폴백)

**보존**: 카카오·네이버 서버 액션 완전 무변. 외부 브라우저/PC 기존 흐름 완전 무변.
